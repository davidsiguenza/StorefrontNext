# StorefrontNext

Personal hub for everything I build around **Salesforce Storefront Next** — toolkits, skills, patterns, and reference demos.

## Contents

| Folder | Description |
|---|---|
| [`sfn-demo-toolkit/`](./sfn-demo-toolkit/) | Toolkit to bootstrap and brand SFN demos for customers end-to-end. Clones the official template, applies a non-invasive branding system, scrapes a customer URL, generates per-client content/theme/profile, and (optionally) generates and imports a product catalog. |
